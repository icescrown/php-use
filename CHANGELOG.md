# 更新日志

本文件记录了 "php use" 扩展的所有重大更改。

## [0.0.5] - 2025-01-20

### 新增
- 支持 PHPDoc 类型检测，自动识别注释中的类引用（@param、@return、@throws、@var、@type）

### 优化
- 优化代码结构，提高可维护性
- 提取正则表达式为类常量，提升性能
- 拆分大方法为多个小方法，提高代码可读性
- 使用现代 JavaScript 语法优化代码
- 优化函数检测性能

### 移除
- 移除常量检测功能（基本用不到）

## [0.0.1] - 2025-12-31

### 新增
- 初始版本发布
- 支持删除未使用的 use 语句
- 支持通过快捷键导入当前类
- 支持展开命名空间（Ctrl+Alt+E）
- 支持自动排序 use 语句功能
- 支持字母顺序和自然顺序两种排序方式
- 增强了代码操作提供程序，支持删除未使用的 use 语句
- 改进了诊断功能，可检测未使用的 use 语句
- 更新了快速修复菜单，根据上下文显示适当的操作

### 功能特性
- 支持单个 `use` 语句和分组 `use` 语句
- 提供快捷键、右键菜单和快速修复选项
- 支持多种可自定义的设置选项
- 自动检测、诊断和通知的配置选项
- 仅针对 PHP 文件，避免意外修改其他类型的文件
