# 更新日志

本文件记录了 "php use" 扩展的所有重大更改。

## [0.0.1] - 2025-12-25

### 新增
- 初始版本发布
- 支持删除未使用的导入
- 支持导入当前类
- 为未导入的类添加了 "导入类" 快速修复选项（使用 Ctrl+. 快捷键）
- 增强了代码操作提供程序，支持删除未使用的导入和导入类
- 改进了诊断功能，可同时检测未使用的导入和未导入的类
- 更新了快速修复菜单，根据上下文显示适当的操作

### 功能特性
- 支持单个 `use` 语句和分组 `use` 语句
- 提供快捷键、右键菜单和快速修复选项
- 支持多种可自定义的设置选项
- 自动检测、诊断和通知的配置选项
- 仅针对 PHP 文件，避免意外修改其他类型的文件
